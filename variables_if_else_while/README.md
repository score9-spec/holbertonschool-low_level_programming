# 0-positive_or_negative
## Description
Ce programme détermine si un nombre est positif, négatif ou nul.

## Utilisation
Pour compiler et exécuter le programme, utilisez les commandes suivantes :
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 0-positive_or_negative.c -o positive_or_negative
./positive_or_negative
